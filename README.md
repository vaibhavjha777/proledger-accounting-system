## 🎯 Overview

**ProLedger** is a powerful, all-in-one accounting and customer management solution designed for small businesses, retail shops, and entrepreneurs. Built entirely with HTML5, CSS3, and vanilla JavaScript, it requires no backend server or database setup - everything runs in your browser with localStorage persistence.

### Key Benefits:
- ✅ **Zero Setup Cost** - No server, no database, no hosting fees
- ✅ **Multi-User Support** - Create multiple accounts with secure authentication
- ✅ **Data Privacy** - All data stored locally in your browser
- ✅ **Responsive Design** - Works on desktop, tablet, and mobile devices
- ✅ **Professional Receipts** - Generate and print/download customer receipts
- ✅ **Real-time Reports** - Daily and monthly financial reports

## ✨ Features

### 🔐 User Management
- **User Registration & Login** - Secure multi-user system
- **Password Protection** - Minimum 4-character password requirement
- **Session Persistence** - Stay logged in across browser sessions
- **User-specific Data** - Each user has their own isolated data

### 💰 Transaction Management
- **Online Transactions** - Track digital/online payments
- **Offline Transactions** - Track in-person/cash payments
- **Customer Tracking** - Unique customer IDs for repeat customers
- **Bulk Operations** - Delete all transactions for a customer

### 📊 Financial Analytics
- **Real-time Dashboard** - Live statistics and metrics
- **Total Items Counter** - Track quantity of items sold
- **Income Breakdown** - Separate online vs offline income
- **Net Profit Calculation** - Income minus expenses
- **Customer Count** - Track unique customers

### 🧾 Professional Receipts
- **Consolidated Receipts** - View complete customer history
- **Print Functionality** - Print receipts directly
- **PNG Download** - Save receipts as images
- **Detailed Transaction History** - All transactions in one place

### 💸 Expense Tracking
- **Record Expenses** - Track business expenses (rent, transport, etc.)
- **Expense Categories** - Categorize different expense types
- **Expense Reports** - View expenses in daily/monthly reports

### 📈 Reporting System
- **Daily Reports** - Today's income, expenses, and net profit
- **Monthly Reports** - Month-wise financial summaries
- **Printable Reports** - Generate printable financial statements
- **Detailed Breakdowns** - Expandable transaction and expense lists

### 🎨 User Interface
- **Modern Design** - Gradient backgrounds and smooth animations
- **Responsive Layout** - Works on all screen sizes
- **Live Clock** - Real-time date and time display
- **Tab Navigation** - Organized feature sections
- **Modal Dialogs** - Clean popup interfaces

## 🛠️ Technologies Used

### Frontend
- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with Flexbox and Grid
- **JavaScript (ES6+)** - Core functionality and DOM manipulation

### Libraries & APIs
- **Google Fonts** - Inter font family
- **Font Awesome 6** - Icon library for visual elements
- **html2canvas** - PNG export functionality

### Storage
- **localStorage API** - Persistent data storage
- **JSON** - Data serialization format
